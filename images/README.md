## Image Optimization Tips

1. **Compress images** to reduce file size:
   - JPG: 70-80% quality
   - Use WebP format when possible (with JPG fallback)

2. **Use appropriate dimensions**:
   - Don't upload images larger than necessary
   - Resize before uploading to save bandwidth

3. **Tools for optimization**:
   - TinyPNG (https://tinypng.com)
   - Squoosh (https://squoosh.app)
   - ImageOptim (Mac)

4. **Favicon generation**:
   - Use https://favicon.io to create favicon from text or image
   - Generate multiple sizes automatically
